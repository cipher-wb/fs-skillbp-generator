快捷键

  • Ctrl + S ：保存
  • Ctrl + F ：查找
  • Ctrl + Z ：回退（暂时屏蔽：存在反序列化问题）
  • Ctrl + C ：复制（支持复制节点+连线，支持跨graph）
  • Ctrl + V ：粘贴（支持粘贴节点ID自动分配，支持跨graph）
  • Ctrl + D ：复制粘贴
  • F ：选择节点，则聚焦选中节点
  • F ：未选择节点，则节点全览
  • A ：节点全览
  • O ：聚焦最初节点
  • [ ：聚焦前一个选择节点
  • ] ：聚焦后一个选择节点
  • Space/空格 ：新增节点
