# CLAUDE.md

本文件为 Claude Code (claude.ai/code) 在此工作目录中提供指导。

## 项目概述

这是一个通用工作目录（草稿箱），用于处理各类杂项任务。并非传统意义上的代码库，没有特定的项目结构。

## 预配置权限

以下命令已在 `.claude/settings.local.json` 中预先允许：
- `npm install:*`
- `claude:*`
- `uv run:*`
- `python:*`

## 语言要求（强制）

- 与用户的对话必须使用简体中文
- 生成的所有文件必须使用简体中文
- 以下情况可使用英文：代码、命令、技术术语、专有名词、文件路径等必须使用英文的部分
